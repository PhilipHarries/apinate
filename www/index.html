apinate

single binary, config-driven, arbitrary APIs

What is apinate?

apinate is a simple HTTP API that can be configured to map any HTTP request to any back end script or command and display the output in a variety of formats.

Simple, clean operation

apinate runs as a single binary, and is driven by a single configuration file. It can output JSON, YAML or plain text, and can output templated HTML.

apinate is the simplest and easiest way of spinning up a functional API.

Write a script to display username and login time of all current system users, then map it to a HTTP endpoint that returns JSON. Then change your mind and return HTML. In under 90 seconds.
